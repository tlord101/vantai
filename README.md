## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Environment Variables

The app uses Firebase and Paystack configurations embedded in the code. For production, move these to environment variables.
